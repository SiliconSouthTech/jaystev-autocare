Welcome to Kaimono Kago

Kaimono Kago is a shopping cart built on Codeigniter. We use BackendPro for the administration and Ajax file manager for file management.

Kaimono Kago features:

  • ADMIN
    • Admin login, forgot password function
    • Admin Categories management
    • Admin Customers management
    • Admin File management
    • Admin Menu management
    • Admin Ajax messages
    • Admin Orders management
    • Admin News letter subscriber management
    • Admin Product management
    • Admin Event calendar
    • Admin Page management
  • CUSTOMERS
    • Email order notice to admin and cutomer
    • jQuery Drop-down site menu
    • Different types of slideshow
    • Customerr News letter subscription
    • Customer Login system
